# Assumptions and Limitations

The workflows use synthetic data and simplified dynamics.

Limitations:
- synthetic data
- simplified dependency network
- simplified threshold failure model
- simplified buffer depletion model
- simplified cascade scoring
- no empirical validation
- no legal, financial, engineering, medical, cybersecurity, public-policy, infrastructure, AI-governance, or safety advice

Professional use requires domain expertise, validated data, stakeholder review, governance, and careful interpretation.
