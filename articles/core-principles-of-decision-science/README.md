# Core Principles of Decision Science

This companion directory supports the article **"Core Principles of Decision Science."**

The workflows operationalize core decision-science principles: structured framing, uncertainty representation, trade-off transparency, normative and descriptive integration, sensitivity analysis, scenario comparison, systems awareness, robustness, adaptability, decision records, and iterative learning.

## Modeling Principles

1. Define the decision before modeling options.
2. Make alternatives explicit and improve the option set before scoring.
3. Represent uncertainty honestly through probabilities, ranges, scenarios, and assumptions.
4. Surface values and trade-offs through criteria, weights, thresholds, and profiles.
5. Combine normative rigor with descriptive realism.
6. Use sensitivity and scenario analysis to test assumption dependence.
7. Account for systems consequences, feedback, delay, and spillovers.
8. Prefer robustness and adaptability under deep uncertainty.
9. Preserve decision records for accountability and learning.
10. Treat computational models as supports for judgment, not replacements for responsibility.

## Quick Start

From this article directory:

```bash
python3 python/run_all_core_principles_workflows.py
Rscript r/run_all_core_principles_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/figures/
outputs/decision_records/
```
