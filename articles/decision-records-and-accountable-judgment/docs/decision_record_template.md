# Decision Record Template

Recommended fields:

- Decision statement
- Decision owner
- Recommendation owner
- Date
- Decision context
- Alternatives considered
- Alternatives rejected or deferred
- Evidence and claim links
- Assumptions
- Uncertainty
- Criteria and trade-offs
- Dissent and unresolved concerns
- Selected action
- Rationale
- Implementation owner
- Monitoring indicators
- Review triggers
- Revision authority
