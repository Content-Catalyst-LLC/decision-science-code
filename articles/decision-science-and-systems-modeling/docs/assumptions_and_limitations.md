# Assumptions and Limitations

The workflows use synthetic system data and simplified dynamics.

Limitations:
- synthetic data
- simplified stock-and-flow logic
- simplified feedback equations
- simplified delay behavior
- simplified scenario performance
- simplified threshold analysis
- no empirical validation
- no legal, financial, engineering, medical, public-policy, climate, organizational, or safety advice

Professional use requires domain expertise, validated data, stakeholder review, governance, and careful interpretation.
