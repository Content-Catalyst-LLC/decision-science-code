# Responsible Use

These workflows are decision-support scaffolds.

Use them to:
- compare infrastructure alternatives
- examine scenario performance
- test lifecycle and resilience trade-offs
- flag equity, environmental, and feasibility concerns
- simulate asset condition and service reliability
- structure decision records
- make assumptions explicit

Do not use them to approve, design, finance, procure, build, operate, or decommission real infrastructure assets without qualified professional review.
