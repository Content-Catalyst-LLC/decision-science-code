# Ecological Thresholds

Some environmental systems can absorb pressure up to a point, then change rapidly after thresholds are crossed.

Relevant concepts:
- safe minimum standards
- precautionary triggers
- irreversibility
- cumulative pressure
- regeneration limits
- threshold uncertainty
- early warning indicators
- planetary boundaries
- system collapse risk

Ecological limits change the logic of decision-making because not all harms can be treated as reversible costs.
