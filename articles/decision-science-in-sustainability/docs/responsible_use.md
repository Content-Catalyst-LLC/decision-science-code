# Responsible Use

These workflows are decision-support scaffolds.

Use them to:
- compare sustainability strategies across objectives
- examine scenario robustness
- flag threshold, equity, and implementation risks
- simulate resource pressure and adaptive response
- document decision records and revision triggers
- support transparent sustainability reasoning

Do not use them to automate high-stakes decisions, manufacture certainty, hide value judgments, bypass affected communities, or replace accountable human judgment.
