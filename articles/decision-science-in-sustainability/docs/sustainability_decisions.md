# Sustainability Decisions

Sustainability decisions differ from conventional decisions because they involve:
- long time horizons
- system interdependence
- ecological thresholds
- deep uncertainty
- multiple stakeholders
- distributed burdens and benefits
- intergenerational consequences
- implementation complexity

The central challenge is not only choosing efficiently in the present, but choosing in ways that remain defensible when delayed effects, system feedback, unequal burdens, and future constraints become visible later.
