# Modeling Principles

## Structure before calculation

A decision tree should clarify the decision frame before expected values are calculated.

## Separate choice from uncertainty

Decision nodes represent actions under control. Chance nodes represent uncertain events outside direct control.

## Terminal values are value judgments

Terminal values may be monetary payoffs, utilities, risk scores, health outcomes, public values, or composite criteria. They should be documented and tested.

## Rollback depends on assumptions

Rollback analysis is only as credible as the tree structure, probabilities, terminal values, and decision rule.
