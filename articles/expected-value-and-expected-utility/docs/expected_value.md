# Expected Value

Expected value is the probability-weighted average of outcomes:

EV = sum(p_i * x_i)

It is useful for:
- repeated decisions
- portfolio reasoning
- risk-neutral comparison
- actuarial logic
- reliability and expected-cost analysis

It is limited when:
- downside consequences are severe
- probabilities are uncertain
- outcomes are not linearly valued
- stakeholder values differ
- consequences are irreversible
