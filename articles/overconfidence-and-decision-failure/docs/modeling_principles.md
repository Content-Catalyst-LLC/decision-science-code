# Modeling Principles

## Confidence must be tested

Confidence should be compared with accuracy, evidence quality, calibration, uncertainty range coverage, and track record.

## Forecasts need scoring

Probabilistic forecasts should be recorded before outcomes and scored after outcomes.

## Ranges need coverage checks

Uncertainty ranges are useful only if actual outcomes fall inside them at the stated frequency.

## Plans need reference classes

Planning estimates should be compared against comparable cases, not only internal narratives.

## Records support accountability

Decision records preserve confidence, assumptions, dissent, uncertainty, review triggers, and outcomes before hindsight rewrites the decision.
