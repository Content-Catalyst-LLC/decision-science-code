# Assumptions and Limitations

The workflows use synthetic data and simplified dynamics.

Limitations:
- synthetic data
- simplified switching-cost accumulation
- simplified lock-in scoring
- simplified option-value decay
- simplified scenario performance
- no empirical validation
- no legal, financial, engineering, medical, public-policy, infrastructure, AI-governance, or safety advice

Professional use requires domain expertise, validated data, stakeholder review, governance, and careful interpretation.
