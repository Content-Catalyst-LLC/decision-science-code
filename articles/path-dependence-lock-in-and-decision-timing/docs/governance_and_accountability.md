# Governance and Accountability

Path-dependent governance should document:
- current path
- alternative paths
- historical commitments
- lock-in mechanisms
- switching costs
- sunk-cost risks
- option value
- timing logic
- critical windows
- exit paths
- stakeholder transition issues
- dissent
- review triggers
- revision authority

Path-dependent accountability means documenting not only what was chosen, but what future constraints the choice creates.
