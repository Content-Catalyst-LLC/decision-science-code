# Responsible Use

These workflows are decision-support scaffolds.

Use them to:
- compare timing and path options
- estimate lock-in risk
- track switching-cost accumulation
- identify option-value decay
- flag review triggers
- document exit paths and timing assumptions
- preserve decision records

Do not use them to automate high-stakes decisions, manufacture certainty, suppress dissent, justify lock-in, or replace accountable human judgment.
