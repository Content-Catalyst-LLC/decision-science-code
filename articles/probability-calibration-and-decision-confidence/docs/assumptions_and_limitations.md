# Assumptions and Limitations

The workflows use synthetic forecast data and stylized calibration diagnostics.

Limitations:
- synthetic probabilities
- simulated outcomes
- simplified confidence profiles
- simplified reference classes
- no empirical validation
- no domain-specific deployment guidance
- no legal, financial, clinical, public-policy, engineering, or safety advice

Professional use requires domain expertise, real outcome data, validation, subgroup audits, governance, and accountable review.
