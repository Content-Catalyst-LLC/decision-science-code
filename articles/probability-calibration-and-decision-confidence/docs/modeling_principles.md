# Modeling Principles

## Confidence should be testable

When confidence is expressed as probability, it can be compared with observed outcomes.

## Calibration is not the same as accuracy

Accuracy asks whether predictions were correct. Calibration asks whether stated probabilities match observed frequencies.

## Base rates matter

Reference classes help anchor probability estimates before case-specific evidence shifts them.

## Thresholds require calibrated probabilities

Decision thresholds become unreliable when the probabilities that trigger them are miscalibrated.

## Decision records make calibration possible

Forecasts must be preserved before outcomes occur if calibration is to be evaluated later.
