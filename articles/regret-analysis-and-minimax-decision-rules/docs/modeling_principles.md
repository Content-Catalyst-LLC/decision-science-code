# Modeling Principles

## Regret is relative performance

Regret measures the gap between the payoff of a chosen action and the best payoff available in the realized state.

## Maximin protects absolute downside

Maximin selects the action with the highest worst-case payoff.

## Minimax regret protects against hindsight loss

Minimax regret selects the action with the smallest worst missed opportunity.

## Expected value requires credible probabilities

Expected value is strongest when probability estimates are defensible.

## Thresholds are still necessary

A low-regret option may still fail minimum performance, safety, equity, or service thresholds.

## Records prevent hindsight distortion

Decision records preserve the scenario set, payoff assumptions, regret matrix, decision rule, selected action, and rationale.
