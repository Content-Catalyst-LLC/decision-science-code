# Resilience, Adaptation, and Long-Horizon Decisions

This companion directory supports the article **"Resilience, Adaptation, and Long-Horizon Decisions."**

The workflows operationalize resilient long-horizon decision-making as a reproducible decision-science problem involving resilience capacity, adaptive recovery, repeated shocks, long-term value, near-term cost, scenario performance, threshold review, intertemporal trade-offs, adaptive pathways, and decision-record scaffolding.

## Core Claim

Long-horizon decisions cannot be evaluated responsibly by short-term performance alone. Decision-makers need strategies that preserve system function, recovery capacity, legitimacy, flexibility, and future decision capacity across uncertainty, disruption, institutional drift, changing values, and extended time horizons.

## Modeling Principles

1. Define the long-horizon decision, time horizon, and affected stakeholders.
2. Identify essential functions that must be preserved under stress.
3. Treat resilience capacity as a stock that can be built, depleted, or exhausted.
4. Compare strategies across stable, stressed, constrained, drifting, and shocked futures.
5. Make intertemporal trade-offs explicit rather than hiding them in short-term scores.
6. Evaluate robustness, adaptability, reversibility, worst-case performance, and threshold pass rates.
7. Define adaptive pathways, monitoring indicators, trigger points, fallback options, and revision authority.
8. Preserve institutional memory through decision records.
9. Treat intergenerational responsibility as part of the decision, not an afterthought.
10. Use models as structured learning tools, not certainty machines.
