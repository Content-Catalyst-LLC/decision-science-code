# Robust Decision-Making

This companion directory supports the article **"Robust Decision-Making."**

The workflows operationalize robust decision-making as a reproducible decision-science problem involving strategy performance across uncertain futures, expected value, worst-case performance, regret, threshold compliance, vulnerability analysis, stress testing, adaptive triggers, strategy durability, and decision-record scaffolding.

## Core Claim

Robust decision-making identifies strategies that remain workable across many plausible futures rather than optimizing for one predicted outcome. It is especially useful when uncertainty is deep, models are fragile, probabilities are contested, and future conditions cannot be reduced honestly to a single forecast.

## Quick Start

```bash
python3 python/run_all_robust_decision_workflows.py
Rscript r/run_all_robust_decision_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/figures/
outputs/decision_records/
```
