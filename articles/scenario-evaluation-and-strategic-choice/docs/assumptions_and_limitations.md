# Assumptions and Limitations

The workflows use synthetic scenario data and simplified performance logic.

Limitations:
- synthetic data
- simplified scenario performance
- simplified regret and robustness scoring
- simplified threshold analysis
- simplified volatility simulation
- no empirical validation
- no legal, financial, engineering, medical, climate, public-policy, organizational, or safety advice

Professional use requires domain expertise, validated data, stakeholder review, governance, and careful interpretation.
