# Key-Driver Diagnostics

Key-driver diagnostics identify the variables that have the largest influence on outcomes or decision rankings.

Useful questions:
- Which input changes the outcome most?
- Which input changes the ranking?
- Which input controls downside exposure?
- Which input is both uncertain and decision-relevant?
- Which input can be monitored?
- Which input can be controlled?

Key drivers should become priorities for evidence, monitoring, and review.
