# Article Notes: Stakeholder Values and Decision Legitimacy

This article explains how decisions become defensible when affected groups disagree about values, evidence, burdens, trade-offs, and acceptable risk.

Core themes:
- stakeholder values
- affectedness
- decision legitimacy
- procedural legitimacy
- substantive legitimacy
- epistemic legitimacy
- trade-offs
- burden analysis
- stakeholder mapping
- multi-criteria decision analysis
- dissent records
- accountability
