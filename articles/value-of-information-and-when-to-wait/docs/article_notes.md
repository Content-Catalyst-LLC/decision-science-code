# Article Notes: Value of Information and When to Wait

This article explains how decision-makers decide whether additional evidence is worth gathering before action and when delay becomes more costly than imperfect choice.

Core themes:
- value of information
- expected value of perfect information
- expected value of sample information
- decision-change probability
- Bayesian updating
- information cost
- delay cost
- opportunity cost
- option value
- irreversibility
- staged action
- adaptive pathways
- decision records
