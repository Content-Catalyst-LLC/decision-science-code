# Assumptions and Limitations

The workflows use synthetic data and simplified value-of-information calculations.

Limitations:
- synthetic data
- simplified probability model
- simplified evidence model
- simplified posterior probabilities
- simplified delay-cost logic
- no empirical validation
- no legal, financial, engineering, climate, medical, policy, organizational, or safety advice

Professional use requires domain expertise, validated evidence, stakeholder review, governance, and careful interpretation.
