# Decision Science Framework

A professional decision science workflow should connect:

1. Decision frame
2. Alternatives
3. Objectives
4. Criteria
5. Evidence
6. Assumptions
7. Scenarios
8. Probabilities
9. Outcomes
10. Values and utilities
11. Trade-offs
12. Sensitivity analysis
13. Regret and robustness
14. Decision record
15. Monitoring and learning

The framework is not designed to automate responsibility. It is designed to make judgment more transparent, structured, and reviewable.
