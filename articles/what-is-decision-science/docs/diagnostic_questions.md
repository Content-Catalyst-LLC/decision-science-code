# Diagnostic Questions

- What decision is being made?
- Who owns the decision?
- What alternatives are available?
- What alternatives were excluded?
- What objectives matter?
- What criteria represent those objectives?
- What evidence supports each assumption?
- Which uncertainties matter most?
- Which assumptions change the recommendation?
- Which option has the highest expected value?
- Which option has the lowest maximum regret?
- Which option is most robust across scenarios?
- Which option is most reversible?
- Which stakeholders bear risk?
- What should be monitored after the decision?
- When should the decision be revisited?
